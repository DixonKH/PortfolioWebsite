# PortfolioWebsite
This is portfolio website by using Nextjs 
=======
This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

**Live Demo:** https://portfolio-website-rust-seven-73.vercel.app  
**Repository:** https://github.com/DixonKH/PortfolioWebsite  

Personal portfolio website built from scratch on Day 1 of my 8-week Next.js professional roadmap.

## Tech Stack
- **Framework:** Next.js 14+ (App Router + Server Components)  
- **Language:** TypeScript  
- **Styling:** Tailwind CSS + Dark Mode (next-themes)  
- **Deployment:** Vercel (automatic CI/CD)  
- **Dev Tools:** Turbopack, ESLint, Prettier  

## Features (Day 1 – Completed)
- Fully responsive layout (mobile ↔ desktop)  
- Smooth fixed navbar with active states  
- Hero section with animated typing effect  
- About, Projects, and Contact sections (static but ready for dynamic data)  
- Dark/Light mode toggle (persisted)  
- SEO-ready metadata & Open Graph tags  
- Perfect Lighthouse scores (100/100 expected)
